# AI简历生成器

## 项目概述
这是一个AI驱动的简历生成器,用户可以输入自己的技术栈和目标公司的招聘要求,系统将生成一份针对目标公司的定制简历。

## 项目信息
- 仓库名称: ai-resume-generator
- 描述: AI驱动的简历生成器 - 根据用户的技术栈和目标公司的招聘要求,自动生成定制简历。
